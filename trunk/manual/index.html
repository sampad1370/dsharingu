DSharingu
ver: 0.13a, 2006/09/30
Written by Davide Pasca (dpasca at gmail.com)
Official site http://kazzuya.com/dsharingu

This software allows two people to remotely watch or administer a remote PC running Windows.
The main feature lies in the ability for either or both the caller or the answerer to interact with the remote's PC.

I wrote this because family's connection in Italy is firewalled by the ISP. I therefore needed a software that allow administering the PC of the calling party.

Main Features

  • Both caller or callee can interact with each other's screen
  • Simple to use
  • Very small executable
  • Open Source (source code can be downloaded for security inspections and improvements)

Warning !

Use good passwords and be really really sure that the remote party is indeed someone that you can trust.
When enabling the Remote can Use My Screen option in the Settings, a remote user is given total control to your machine.

Requirements for a smooth usage

  • Windows XP/2000 (currently only tested on Windows XP).
  • A decent graphic chip with basic OpenGL support (better if you have recent video-card drivers anyway !).
  • A non-ancient system: 1GHz CPU and 512MB of RAM.
  • DSL/Cable connection or better.

Known issues

  • Currently, interaction is only possible throught mouse. Keyboard support will come later.
  • Rather high bandwidth is still required for good interaction.
