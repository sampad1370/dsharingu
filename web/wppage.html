DSharingu - Remote Desktop Sharing software

DSharingu is a desktop/screen sharing application.
It's made to be simple and to minimize the firewall issues.
With DSharingu there is no client and server concept. This program allows for either or both caller and callee to administer the other computer remotely.
I created this software to help my family in Italy with their computer issues. Their Internet provider doesn't allow to accept incoming connections and so VNC couldn't be used.

>> Download DSharingu from Google Code <<

Features:

  • Both caller and callee can control each other's PC
  • Simple to use
  • Very small executable (only about 150KB)
  • Open Source: free to use and redistribute. See License.txt
  • Automatic call at startup
  • One button update
  • Handles multiple connection with tabbed view interface

Requirements:

  • Windows XP
  • 1GHz CPU or better
  • OpenGL accelerated graphic chip (get the latest drivers !)
  • Luck !

Known Problems:

  • Remote control doesn't yet support keyboard
  • Many blatant bugs. This project is in a very alpha development stage
