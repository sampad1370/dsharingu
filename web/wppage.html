DSharingu is a desktop/screen sharing application. It's made to be simple and to minimize the firewall issues.
With DSharingu there is no client and server concept.
In some cases one wants to remotely administer a computer which can't accept connection. This program will allow either side to see or administer each other.
I created this program so that I can help my family in Italy with their computer issues. Their Internet provider doesn't allow to accept incoming connections and so VNC couldn't be used.

The source code for this software is available for download. If you are interested, development updates and links to obtain source code are at the following site: http://pascacode.blogspot.com

Features:

  • Both caller and callee can control each other's PC.
  • Automatic call of a remote at startup.
  • Run-time Internet update.
  • Silent install when doing an update. Important when one is updating from remote. Works well in conjunction with the Automatic call feature
  • Multiple connection with tabbed view interface.
  • Simple inteface with the Home Page's tab view for quick access.
  • Open Sourced project: code available for public scrutiny.

Requirements:

  • Windows XP/2000
  • 1GHz CPU or better
  • OpenGL accelerated graphic chip (get the latest drivers !)
  • Luck !

Known Problems:

  • Display quality is currently limited to black&white for performance reasons.
  • Remote control doesn't yet support keyboard.
  • Many blatant bugs. This project is in a very alpha development stage.
